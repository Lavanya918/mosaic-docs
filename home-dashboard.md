# MOSAIC Home Page

## Overview

The **MOSAIC Home Page** is the primary landing dashboard displayed after a user logs into the platform. It provides quick access to the most frequently used workflows, including creating new orders, viewing existing orders, and managing patient information.

MOSAIC (Mammary Oncology Spatial Analysis and Intelligent Classification) supports precision care in breast oncology through spatial analysis and intelligent classification.

## Home Page Layout

The Home Page consists of the following key sections:

1. Top Navigation Bar  
2. Left Navigation Menu  
3. Welcome Banner  
4. Quick Actions  

## Top Navigation Bar

The top navigation bar is visible across the platform and provides workspace- and account-level information.

### Components

  **MOSAIC Logo**
  - Displays MOSAIC branding.
  - Clicking the logo navigates back to the Home Page.

  **Credits Indicator**
  - Displays the number of available credits for running analyses and orders.

  **Workspace Indicator**
  - Shows the currently active workspace.

  **Notifications Icon**
  - Displays system notifications and alerts.

  **User Profile Icon**
  - Opens user-related actions such as account settings and sign out.


## Left Navigation Menu

The left sidebar allows navigation to the main sections of the platform.

### Menu Items

- **Home**
  - Opens the Home Page dashboard.

- **Orders**
  - Navigates to order-related workflows such as creating and managing orders.

- **Patients**
  - Opens patient management and record-related features.

- **Model**
  - Provides read-only information about the MOSAIC model.

The active menu item is visually highlighted.

## Quick Actions Section

The **Quick Actions** section provides visual cards that allow users to quickly access commonly used workflows.

### Available Quick Actions

#### Create New Order
- Available as a card in the Quick Actions section.
- Provides an alternative entry point to start a new order.
- Leads to the **same Create New Order workflow** as the primary button in the Welcome Banner.

#### View All Orders
- Opens the complete list of existing and previous orders.

#### Patients
- Opens the patient management section.

Each Quick Action card includes a short description and a navigation arrow for ease of use.

## Typical User Workflow

1. User logs into MOSAIC.
2. Reviews available credits and workspace details.
3. Initiates one of the following actions:
   - Create a new order
   - View existing orders
   - Manage patient records
